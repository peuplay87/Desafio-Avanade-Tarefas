# 📝 Gerenciador de Tarefas Fullstack

Projeto desenvolvido para o desafio técnico da Avanade.

## 🚀 Tecnologias
- **Back-end:** .NET 8 / Entity Framework / SQLite
- **Front-end:** Angular 17 / Bootstrap

## ⚙️ Como executar
1. No terminal da `TarefasApi`, rode `dotnet run`.
2. No terminal da `TarefasFront`, rode `ng serve`.
3. Acesse `http://localhost:4200`.
